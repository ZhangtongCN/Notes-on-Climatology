# 气候学 课程学习笔记
NUIST 气候学 课程学习笔记

**包含内容**： 第一章 绪论、第二章 气候系统、第三章 气候系统的能量平衡、第四章 气候系统的水分过程、第五章 大气系统的基本状况、第六章 海气相互作用、第七章 陆-气相互作用、第八章 冰-气相互作用、第九章 气候变化。

**适配教材**：气象出版社, 廖启龙等,《现代气候学(第二版)》

**适配课程**：南京信息工程大学 现代气候学
